# Prayer Time Widget

A Notion-embeddable prayer times widget with Islamic wallpaper, Apple-style UI, and live location support.

## Structure

- `index.html` — main entry point
- `js/prayer-widget.js` — widget logic
- `style.css` — Apple native UI styles
- `assets/wallpapers/` — Islamic background

## Setup

1. Open `index.html` in a browser
2. Enter your free API key from [islamicapi.com](https://www.islamicapi.com)
3. Save settings and use **Copy embed link** to paste into Notion
